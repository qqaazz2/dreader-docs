## Windows 安装

Windows 版本以 **ZIP 压缩包** 形式提供。

1. 下载 Windows 对应版本的 ZIP 文件  
2. 将压缩包解压到任意目录  
3. 双击运行 `dreader.exe` 即可启动 **DReader**

![解压后Windows下的目录结构](/assets/image/windows-setup.png "解压后Windows下的目录结构")

## Android 安装

Android 版本以 **APK 文件** 形式提供。

1. 将 APK 文件下载到 Android 设备  
2. 打开 APK 文件并按照系统提示完成安装  
3. 安装完成后即可正常使用 **DReader**

> 若系统提示“禁止安装未知来源应用”，  
> 请在系统设置中临时允许本次安装。
